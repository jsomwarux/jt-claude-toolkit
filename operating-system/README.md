# operating-system

JT-specific workflows mirrored from OpenClaw's live source of truth.

Use this plugin as a portable projection layer. The canonical rules remain in the OpenClaw
workspace:

- `AGENTS.md` for hard operating/security rules.
- `TOOLS.md` for commands and paths.
- `MEMORY.md` for current decisions/status.
- `skills/` for repeatable workflows.
- `agents/` for recurring/stateful operators.
